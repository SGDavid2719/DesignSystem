## Descripción general

Este componente es un átomo ya que es una unidad indivisible. <br />
Un botón es un elemento interactivo que se define con la etiqueta `<button>`, y al cual es necesario añadir atributos como `onClick` para especificar lo que debe ocurrir cuando se hace click.
