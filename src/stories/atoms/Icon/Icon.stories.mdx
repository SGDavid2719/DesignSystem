## Desarrollo atómico

Al ser una unidad indivisible se utiliza únicamente la etiqueta HTML propia como es `<svg>` y `<path>`.

```jsx
<svg xmlns="http://www.w3.org/2000/svg" viewBox={iconId} fill="currentColor">
    <path fillRule="evenodd" d clipRule="evenodd" />
</svg>
```
