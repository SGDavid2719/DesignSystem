## Desarrollo atómico

Este componente es un organismo y está compuesto por moléculas como una lista de `<MenuLink>`. <br />
Todo el menú está compuesto por componentes del menú que son a su vez redirecciones.

```jsx
<div>
    {openned && (
        <ul>
            {menuLinks.map((menuLink) => {
                const { href, ariaLabel, mainText, subText, rel } = menuLink;
                return (
                    <li key>
                        <MenuLink rel href ariaLabel mainText subText />
                    </li>
                );
            })}
        </ul>
    )}
    <div>
        <Button
            label="FUEIB"
            renderIcon="add"
            align="start"
            onClick={() => setOppened((prevState) => !prevState)}
        />
    </div>
</div>
```
