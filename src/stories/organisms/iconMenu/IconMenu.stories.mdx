## Desarrollo atómico

Este componente es un organismo y está compuesto por una lista de moléculas como es `<IconLink>`. <br />
Los iconos de la lista son también enlaces.

```jsx
<ul>
    {iconLinks.map((iconLink) => {
        const { href, ariaLabel, iconClassName, iconId, rel } = iconLink;
        return (
            <li key>
                <IconLink rel href ariaLabel iconClassName iconId />
            </li>
        );
    })}
</ul>
```
