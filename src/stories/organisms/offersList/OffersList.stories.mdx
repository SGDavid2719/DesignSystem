## Desarrollo atómico

Este componente es un organismo y está compuesto por moléculas como una lista de `<OfferLink>`. <br />
Para hacer accesible la lista en vez de hacer scroll automático se ha habilitado el scroll manual.

```jsx
<div>
    <div>
        <Span children />
        <Icon iconId />
    </div>
    <div>
        {offerLinks.map((offerLink) => {
            return (
                <div key>
                    <OfferLink {...offerLink} />
                </div>
            );
        })}
    </div>
    <div>
        <Redirect {...redirect} />
    </div>
</div>
```
