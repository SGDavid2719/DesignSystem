## Desarrollo atómico

Este componente es un organismo y está compuesto por moléculas como una lista de `<MenuLink>`

```jsx
<>
    <Button
        label="MENÚ"
        renderIcon="menu"
        align="start"
        onClick={() => setShowMenu((prevState) => !prevState)}
    />
    {showMenu &&
        ReactDOM.createPortal(
            <div>
                {sections.map((section, sectionIndex) => {
                    const { title, sectionLinks } = section;

                    return (
                        <div key>
                            <h1>
                                {title}
                            </h1>
                            <ul>
                                {sectionLinks.map((sectionLink, sectionLinkIndex) => {
                                    const {
                                        href,
                                        ariaLabel,
                                        mainText,
                                        subText,
                                        rel,
                                    } = sectionLink;

                                    return (
                                        <li key>
                                            <MenuLink
                                                rel
                                                href
                                                ariaLabel
                                                mainText
                                                subText
                                            />
                                        </li>
                                    );
                                })}
                            </ul>
                        </div>
                    );
                })}
            </div>,
            portalElement!
        )}
    {showMenu && <Backdrop onOutsideClick={() => setShowMenu(false)} />}
</>
```
