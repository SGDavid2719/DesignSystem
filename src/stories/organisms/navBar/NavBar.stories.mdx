## Desarrollo atómico

Este componente es un organismo y está compuesto por moléculas como `<MenuButton>` y una lista de `<MenuLink>`. <br />
Es necesario incluir el `<div>` con id `overlays` ya que el menú desplegado se mostrará ahí.

```jsx
<>
    <div>
        <Button
            renderIcon="menu"
            align="start"
            onClick={() => setShowMenu((prevState) => !prevState)}
        />
    </div>
    <nav>
        <ul>
            {mainLinks.map((menuLink) => {
                const { href, ariaLabel, mainText, subText, subSections, rel } =
                    menuLink;
                return (
                    <li key>
                        <MenuLink
                            rel
                            href
                            ariaLabel
                            mainText
                            subText
                            subSections
                        />
                    </li>
                );
            })}
            <li>
                <MenuButton sections />
            </li>
        </ul>
    </nav>
    <div id="overlays" />
</>
```
