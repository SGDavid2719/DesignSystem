## Descripción general

Componente para mostrar la información relacionada con una oficina, como el nombre, localización, número de teléfono, correo y horarios.

## Desarrollo atómico

Esta molécula está compuesta por contenedores en línea e iconos.

```jsx
<div>
    <div>{name}</div>
    <div>
        <div>
            <Icon iconId="location" />
            <Span children={location} />
        </div>
    </div>
    <div>
        <div>
            <Icon iconId="phone" />
            <Span children={phoneNumber} />
        </div>
    </div>
    <div>
        <div>
            <Icon iconId="mail" />
            <Span children={email} />
        </div>
    </div>
    <div>
        <div>
            <Icon iconId="schedule" />
            {schedules.map((schedule, index) => (
                <Span key children={schedule} />
            ))}
        </div>
    </div>
</div>
```
