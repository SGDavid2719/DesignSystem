## Desarrollo atómico

Al ser una molécula, este componente está compuesto por dos o varios átomos. <br />
En este caso, está compuesto por contenedores en línea e iconos.

```jsx
<div>
    <div>{name}</div>
    <div>
        <Span
            children={
                <>
                    <Icon iconId="location" />
                    {location}
                </>
            }
        />
    </div>
    <div>
        <Span
            children={
                <>
                    <Icon iconId="phone" />
                    {phoneNumber}
                </>
            }
        />
    </div>
    <div>
        <Span
            children={
                <>
                    <Icon iconId="mail" />
                    {email}
                </>
            }
        />
    </div>
    <div>
        <Span
            children={
                <>
                    <Icon iconId="schedule" />
                    {schedules.map((schedule, index) => (
                        <Span key children={schedule} />
                    ))}
                </>
            }
        />
    </div>
</div>
```
