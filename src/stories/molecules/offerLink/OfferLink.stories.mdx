## Desarrollo atómico

Al ser una molécula, este componente está compuesto por dos o varios átomos. <br />
En este caso, se trata de un enlace y un icono.

```jsx
<div>
    <div>
        <Redirect {...offerRedirect} />
    </div>
    {(firstSubtitle || secondSubtitle) && (
        <div className="flex gap-1">
            <Span children={firstSubtitle} /> -{" "}
            <Span children={secondSubtitle} />
        </div>
    )}
    <div>
        <Span
            children={
                <>
                    <Icon iconId="phone" />
                    {offerName}
                </>
            }
        />
    </div>
    <div>
        <Span
            children={
                <>
                    <Icon iconId="location" />
                    {placeName}
                </>
            }
        />
    </div>
    <div>
        <Span
            children={
                <>
                    <Icon iconId="schedule" />{" "}
                    {`Hace ${dateDiffInDays(new Date(), publicationDate)} días`}{" "}
                </>
            }
        />
    </div>
    <div>
        <Span
            children={
                <>
                    <Icon iconId="mail" />
                    {`Ref. ${referenceNumber}`}
                </>
            }
        />
    </div>
</div>
```
