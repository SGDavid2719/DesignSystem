## Desarrollo atómico

Esta molécula se trata de una lista, con items de tipo enlace. <br />
Además, después de cada enlace exceptuando el último, se añade un punto.

```jsx
<ul>
    {links.map((link, index) => {
        const { href, ariaLabel, children, rel } = link;
        return (
            <Fragment key>
                <li>
                    <Redirect rel href ariaLabel children />
                </li>
                {index + 1 !== links.length && (
                    <li>
                        <Span children="·" />
                    </li>
                )}
            </Fragment>
        );
    })}
</ul>
```
