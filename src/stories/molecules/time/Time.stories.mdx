Este componente es una molécula ya que está compuesto por dos o más átomos. <br />
Un elemento `<time>` se utiliza para definir un `datetime` específico que pueda ser traducido por un formateador por máquina.

## Desarrollo atómico

Este componente es una molécula y está compuesto por átomos `<span>` y `<time>` con el objetivo de hacerlo más visual.

```jsx
<time dateTime>
    <Span children={day} />
    <Span children={month} />
</time>
```
