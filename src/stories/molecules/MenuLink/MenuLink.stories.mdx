## Desarrollo atómico

Esta molécula está formada de un enlace, contenedores en línea y una lista de enlaces.

```jsx
<div onMouseEnter onMouseLeave>
    <Redirect rel href ariaLabel>
        <Span children={mainText} />
        <Span children={subText} />
    </Redirect>
    {subSections && mouseEnter && (
        <ul>
            {subSections?.map((subSection) => {
                const { href, ariaLabel, children, rel } = subSection;
                return (
                    <li key>
                        <Redirect rel href ariaLabel children />
                    </li>
                );
            })}
        </ul>
    )}
</div>
```
